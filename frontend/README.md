# WECARE Frontend (Rebuilt)

## Run frontend

```bash
cd frontend
node server.js
```

Frontend URL: `http://localhost:5500`

## Main pages

- `index.html` - public project website (problem, objectives, methodology, modules)
- `login.html` - sign in page with OAuth provider discovery
- `signup.html` - healthcare provider signup
- `dashboard.html` - role-aware WECARE operations dashboard
- `oauth-callback.html` - OAuth token handoff page

## API expected

- `http://localhost:5000/api`

## Core modules from write-up

1. User Authentication Module
2. Student Health Records Database
3. Health Monitoring Dashboard
4. Emergency Alert Workflow
5. Communication and Notification Module
